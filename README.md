# Vehicle IoT Sensor Simulator

A Python-based project that simulates vehicle telemetry data and sends it to a Flask backend API.

## Current Features
- Simulates vehicle speed, engine temperature, and fuel level
- Sends telemetry data to a backend using HTTP
- Receives and stores data in a Flask server
- Supports live data flow for future dashboard and cloud expansion

## Tech Used
- Python
- Flask
- Requests
- GitHub Codespaces

## Next Steps
- Add dashboard visualization
- Store data in a database
- Connect to cloud services
- Expand toward ESP32 or embedded hardware integration
